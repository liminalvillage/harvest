## Getting Started

### Prerequisites
- Node.js (v20 or higher)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/liminalvillage/harvest.git
```

2. Install dependencies:

```bash
yarn 
```

3. Run the development server:

```bash
yarn dev
```
